Riverpod Generator

Riverpod Generator adalah alat yang membantu mengotomatisasi pembuatan provider dalam Riverpod, sebuah state management library untuk Flutter. Ini memungkinkan kita untuk menulis provider dengan lebih sedikit boilerplate code menggunakan anotasi.

AsyncNotifierProvider adalah jenis provider yang digunakan untuk mengelola state asynchronous. Ini sangat berguna untuk operasi yang memerlukan waktu, seperti mengambil data dari API atau database.

Dengan menggunakan anotasi @riverpod, kita dapat membuat AsyncNotifierProvider dengan lebih mudah. Berikut adalah beberapa poin penting:

  1. Anotasi @riverpod ditempatkan di atas class yang meng-extend AsyncNotifier.
  2. Class tersebut harus memiliki method build() yang mengembalikan Future dari tipe state yang dikelola.
  3. Kita dapat menambahkan method lain untuk memodifikasi state.

Studi Kasus: Shopping Cart​

Sekarang, mari kita buat contoh studi kasus untuk keranjang produk menggunakan Freezed untuk model Product dan json_annotation untuk serialisasi JSON.

Diagram Alur Proses Shopping Cart​

Penjelasan diagram:

  1. Aplikasi dimulai dan pengguna membuka ProductListPage.
  2. Di ProductListPage, pengguna memiliki beberapa pilihan:
    • Melihat produk
    • Menambahkan produk ke keranjang
    • Mengatur kuantitas produk
    • Membuka CartPage
  3. Setiap kali pengguna melakukan aksi yang berhubungan dengan keranjang (menambah, mengubah kuantitas), CartProvider diperbarui dan badge keranjang di-update.
  4. Ketika pengguna membuka CartPage, mereka dapat:
    • Mengatur kuantitas item di keranjang
    • Menghapus item dari keranjang
    • Kembali ke ProductListPage untuk lanjut berbelanja
    • Melanjutkan ke proses checkout (tidak diimplementasikan dalam contoh kode kita)
  5. CartProvider bertanggung jawab untuk mengelola state keranjang, termasuk menambah produk, mengupdate kuantitas, menghapus produk, dan menghitung total.
  6. Setiap perubahan pada CartProvider akan memicu pembaruan UI di ProductListPage (badge keranjang) dan CartPage (daftar item dan total).

Diagram ini memberikan gambaran visual yang jelas tentang alur aplikasi dan bagaimana berbagai komponen berinteraksi satu sama lain. Ini membantu dalam memahami arsitektur aplikasi dan aliran data antara berbagai bagian aplikasi.

5. Buat chart provider​

lib/providers/chart_provider.dart
import 'package:riverpod_annotation/riverpod_annotation.dart';
import '../models/product.dart';
import '../core/result.dart';

part 'cart_provider.g.dart';

@riverpod
class Cart extends _$Cart {
  @override
  Future<Result<List<Product>>> build() async {
    try {
      // Simulate fetching initial cart data from an API or database
      await Future.delayed(Duration(seconds: 1));
      return Result.success([]);
    } catch (e) {
      return Result.failed("Failed to initialize cart: $e");
    }
  }

  Future<Result<List<Product>>> addProduct(Product product) async {
    try {
      final currentResult = await future;
      if (currentResult.isFailed) {
        return Result.failed(currentResult.errorMessage!);
      }
      final currentProducts = currentResult.resultValue!;
      final existingProductIndex =
          currentProducts.indexWhere((p) => p.id == product.id);

      if (existingProductIndex != -1) {
        // Product already exists, increase quantity
        final updatedProducts = [...currentProducts];
        updatedProducts[existingProductIndex] =
            updatedProducts[existingProductIndex].copyWith(
          quantity: updatedProducts[existingProductIndex].quantity + 1,
        );
        state = AsyncValue.data(Result.success(updatedProducts));
        return Result.success(updatedProducts);
      } else {
        // Add new product with quantity 1
        final updatedProducts = [
          ...currentProducts,
          product.copyWith(quantity: 1)
        ];
        state = AsyncValue.data(Result.success(updatedProducts));
        return Result.success(updatedProducts);
      }
    } catch (e) {
      return Result.failed("Failed to add product: $e");
    }
  }

  Future<Result<List<Product>>> removeProduct(String productId) async {
    try {
      final currentResult = await future;
      if (currentResult.isFailed) {
        return Result.failed(currentResult.errorMessage!);
      }
      final currentProducts = currentResult.resultValue!;
      final updatedProducts =
          currentProducts.where((p) => p.id != productId).toList();
      state = AsyncValue.data(Result.success(updatedProducts));
      return Result.success(updatedProducts);
    } catch (e) {
      return Result.failed("Failed to remove product: $e");
    }
  }

  Future<Result<List<Product>>> updateQuantity(
      String productId, int newQuantity) async {
    try {
      final currentResult = await future;
      if (currentResult.isFailed) {
        return Result.failed(currentResult.errorMessage!);
      }
      final currentProducts = currentResult.resultValue!;
      List<Product> updatedProducts;
      if (newQuantity <= 0) {
        // Remove the product if quantity is 0 or less
        updatedProducts =
            currentProducts.where((p) => p.id != productId).toList();
      } else {
        updatedProducts = currentProducts.map((p) {
          if (p.id == productId) {
            return p.copyWith(quantity: newQuantity);
          }
          return p;
        }).toList();
      }
      state = AsyncValue.data(Result.success(updatedProducts));
      return Result.success(updatedProducts);
    } catch (e) {
      return Result.failed("Failed to update quantity: $e");
    }
  }

  Future<Result<double>> getTotalPrice() async {
    try {
      final currentResult = await future;
      if (currentResult.isFailed) {
        return Result.failed(currentResult.errorMessage!);
      }
      final products = currentResult.resultValue!;
      final total = products.fold(
          0.0, (sum, product) => sum + (product.price * product.quantity));
      return Result.success(total);
    } catch (e) {
      return Result.failed("Failed to calculate total price: $e");
    }
  }
}

Penjelasan tentang kode di atas:

  1. Kita menggunakan anotasi @riverpod untuk membuat AsyncNotifierProvider.
  2. Class Cart meng-extend \_$Cart yang akan di-generate oleh Riverpod Generator.
  3. Method build() mengembalikan Future<List<Product>> yang merepresentasikan state awal keranjang.
  4. Kita menambahkan beberapa method untuk memanipulasi keranjang:
    • addProduct(): Menambahkan produk ke keranjang atau meningkatkan kuantitas jika sudah ada.
    • removeProduct(): Menghapus produk dari keranjang.
    • updateQuantity(): Mengubah kuantitas produk dalam keranjang.
    • getTotalPrice(): Menghitung total harga semua produk dalam keranjang.
  5. Setiap method menggunakan state untuk memperbarui dan membaca data keranjang.
  6. Kita menggunakan AsyncValue untuk menangani loading state dan error.

6. Buat product list page​

lib/pages/product_list_page.dart
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import '../providers/cart_provider.dart';
import '../data/dummy_products.dart';
import '../core/result.dart';

class ProductListPage extends ConsumerWidget {
  const ProductListPage({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    return Scaffold(
      appBar: AppBar(
        title: const Text('Product List'),
        actions: [
          Stack(
            alignment: Alignment.center,
            children: [
              IconButton(
                icon: const Icon(Icons.shopping_cart),
                onPressed: () {
                  Navigator.pushNamed(context, '/cart');
                },
              ),
              Positioned(
                right: 8,
                top: 8,
                child: Consumer(
                  builder: (context, ref, _) {
                    final cartAsync = ref.watch(cartProvider);
                    return cartAsync.when(
                      data: (result) {
                        if (result.isSuccess) {
                          int totalItems = result.resultValue!
                              .fold(0, (sum, item) => sum + item.quantity);
                          return Badge(
                            label: Text('$totalItems'),
                            isLabelVisible: totalItems > 0,
                          );
                        } else {
                          return const SizedBox.shrink();
                        }
                      },
                      loading: () => const SizedBox.shrink(),
                      error: (_, __) => const SizedBox.shrink(),
                    );
                  },
                ),
              ),
            ],
          ),
        ],
      ),
      body: ListView.builder(
        itemCount: dummyProducts.length,
        itemBuilder: (context, index) {
          final product = dummyProducts[index];
          return Card(
            margin: const EdgeInsets.all(8),
            child: Padding(
              padding: const EdgeInsets.all(16),
              child: Column(
                crossAxisAlignment: CrossAxisAlignment.start,
                children: [
                  Text(product.name,
                      style: Theme.of(context)
                          .textTheme
                          .titleLarge), // Updated from headline6
                  const SizedBox(height: 8),
                  Text('\$${product.price.toStringAsFixed(2)}',
                      style: Theme.of(context)
                          .textTheme
                          .titleMedium), // Updated from subtitle1
                  const SizedBox(height: 16),
                  Row(
                    children: [
                      ElevatedButton(
                        child: const Text('Add to Cart'),
                        onPressed: () async {
                          final result = await ref
                              .read(cartProvider.notifier)
                              .addProduct(product);
                          if (result.isSuccess) {
                            ScaffoldMessenger.of(context).showSnackBar(
                              SnackBar(
                                  content:
                                      Text('${product.name} added to cart')),
                            );
                          } else {
                            ScaffoldMessenger.of(context).showSnackBar(
                              SnackBar(
                                  content: Text(
                                      'Failed to add to cart: ${result.errorMessage}')),
                            );
                          }
                        },
                      ),
                      const SizedBox(width: 16),
                      Consumer(
                        builder: (context, ref, _) {
                          final cartAsync = ref.watch(cartProvider);
                          return cartAsync.when(
                            data: (result) {
                              if (result.isSuccess) {
                                final cartItem = result.resultValue!.firstWhere(
                                  (item) => item.id == product.id,
                                  orElse: () => product.copyWith(quantity: 0),
                                );
                                return Row(
                                  children: [
                                    IconButton(
                                      icon: const Icon(Icons.remove),
                                      onPressed: cartItem.quantity > 0
                                          ? () => ref
                                              .read(cartProvider.notifier)
                                              .updateQuantity(product.id,
                                                  cartItem.quantity - 1)
                                          : null,
                                    ),
                                    Text('${cartItem.quantity}'),
                                    IconButton(
                                      icon: const Icon(Icons.add),
                                      onPressed: () => ref
                                          .read(cartProvider.notifier)
                                          .updateQuantity(product.id,
                                              cartItem.quantity + 1),
                                    ),
                                  ],
                                );
                              } else {
                                return Text('Error: ${result.errorMessage}');
                              }
                            },
                            loading: () => const CircularProgressIndicator(),
                            error: (_, __) => const Text('Error'),
                          );
                        },
                      ),
                    ],
                  ),
                ],
              ),
            ),
          );
        },
      ),
    );
  }
}

7. Buat chart page​

lib/pages/chart_page.dart
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import '../providers/cart_provider.dart';
import '../core/result.dart';

class CartPage extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final cartAsync = ref.watch(cartProvider);

    return Scaffold(
      appBar: AppBar(
        title: Text('Shopping Cart'),
      ),
      body: cartAsync.when(
        data: (result) {
          if (result.isSuccess) {
            final cart = result.resultValue!;
            if (cart.isEmpty) {
              return Center(child: Text('Your cart is empty'));
            }
            return ListView.builder(
              itemCount: cart.length,
              itemBuilder: (context, index) {
                final product = cart[index];
                return ListTile(
                  title: Text(product.name),
                  subtitle: Text(
                      '\$${(product.price * product.quantity).toStringAsFixed(2)}'),
                  trailing: Row(
                    mainAxisSize: MainAxisSize.min,
                    children: [
                      IconButton(
                        icon: Icon(Icons.remove),
                        onPressed: () async {
                          await ref
                              .read(cartProvider.notifier)
                              .updateQuantity(product.id, product.quantity - 1);
                        },
                      ),
                      Text('${product.quantity}'),
                      IconButton(
                        icon: Icon(Icons.add),
                        onPressed: () async {
                          await ref
                              .read(cartProvider.notifier)
                              .updateQuantity(product.id, product.quantity + 1);
                        },
                      ),
                    ],
                  ),
                );
              },
            );
          } else {
            return Center(child: Text('Error: ${result.errorMessage}'));
          }
        },
        loading: () => Center(child: CircularProgressIndicator()),
        error: (error, stack) => Center(child: Text('Error: $error')),
      ),
      bottomNavigationBar: cartAsync.when(
        data: (result) => Container(
          padding: EdgeInsets.all(16),
          child: Consumer(
            builder: (context, ref, _) {
              return FutureBuilder<Result<double>>(
                future: ref.read(cartProvider.notifier).getTotalPrice(),
                builder: (context, snapshot) {
                  if (snapshot.connectionState == ConnectionState.waiting) {
                    return CircularProgressIndicator();
                  }
                  if (snapshot.hasError) {
                    return Text('Error: ${snapshot.error}');
                  }
                  if (snapshot.hasData) {
                    final result = snapshot.data!;
                    if (result.isSuccess) {
                      return Text(
                        'Total: \$${result.resultValue!.toStringAsFixed(2)}',
                        style: Theme.of(context).textTheme.titleLarge,
                      );
                    } else {
                      return Text('Error: ${result.errorMessage}');
                    }
                  }
                  return SizedBox.shrink();
                },
              );
            },
          ),
        ),
        loading: () => SizedBox.shrink(),
        error: (_, __) => SizedBox.shrink(),
      ),
    );
  }
}

8. Ubah main.dart​

main.dart
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'pages/product_list_page.dart';
import 'pages/cart_page.dart';

void main() {
  runApp(
    ProviderScope(
      child: MyApp(),
    ),
  );
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Shopping App',
      theme: ThemeData(
        primarySwatch: Colors.blue,
        visualDensity: VisualDensity.adaptivePlatformDensity,
      ),
      initialRoute: '/',
      routes: {
        '/': (context) => const ProductListPage(),
        '/cart': (context) => CartPage(),
      },
    );
  }
}
